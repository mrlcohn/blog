Getting started with the blog - foundations and goals

Leonard Cohn

June 11, 2024

Welcome to my very first blog post!

My name is Leonard. I am a DevOps engineer who wants to build projects to develop my skills and learn more about the technologies I use at work and those that are popular in the industry.

In my current position, my team uses the MERN stack, and I sometimes feel intimidated by the codebase. I want to learn more about building and deploying this stack so I can jump in and learn more about the products I am supporting. We use AWS for some of our development and QA deployments, but our production deployments are bare-metal. I want to explore AWS more.

For my first project, I am going to set up a blog that I can use to document my projects and hone my DevOps skills. The blog will involve the following components:

  1. MongoDB
    • Deploy
      • AWS DocumentDB
      • AWS MongoDB Atlas integration
      • Self-hosted DB hosted on AWS infrastructure
    • Document structure of blog posts and users
    • CRUD operations for blog posts
  2. Express.js
    • API to fetch data from database
    • Authorization and authentication
  3. React.js
    • Routing to homepage, login, upload, and blog post pages
    • Integration of Material UI components
    • Basic style (CSS) configurations
    • Vite.js (used by my current employer)
  4. Node.js
  5. DevOps
    • Use as many AWS tools as possible
    • Implement Infrastructure-as-Code
      • Terraform
      • AWS CloudFormation
    • Implement Configuration-as-Code
      • Ansible
      • Chef
      • Puppet
      • AWS
    • CI/CD pipeline management
      • AWS tools
      • Jenkins server
    • Test integration
    • Hosting through Docker and AWS ECS or EKS
    • Branching strategy and GitHub repository

That list is fairly long but doable. I wanted to get the major goals out in the open so that I do not encounter too much scope creep right away. I have other projects in my head that I also want to tackle that will expand on these concepts. I plan to write at least one post as I implement each of these concepts.
